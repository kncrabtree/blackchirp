Python Module Examples
======================

Two end-to-end notebooks accompany the ``blackchirp`` package: a CP-FTMW
walkthrough that loads a single-FID acquisition and explores the FID
processing surface, and a LIF walkthrough that loads a partial scan and
exercises the per-trace and aggregating-helper APIs. Both notebooks are
committed in fully-executed form so the rendered output is visible
inline without re-running them.
