# Changelog

The changelog is maintained in the Sphinx documentation, which is the
source of truth. It is organized per release, newest first:

**https://blackchirp.readthedocs.io/page/changelog.html**

Each release has its own page (e.g. `2.0.0`, `1.1.0`) with a summary,
highlights, component-level changes, and bug fixes. See also the
[migration guide](https://blackchirp.readthedocs.io/page/migration.html)
for upgrading between major versions.

Documentation for a specific release line is available at
`https://blackchirp.readthedocs.io/<version>/` (for example,
`https://blackchirp.readthedocs.io/1.1.x/`).
